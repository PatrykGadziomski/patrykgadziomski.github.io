COVID 19 Dashboard

Project Details

This project analyzes the data on COVID-19 deaths and infections from February 24, 2020, until September 06, 2022.

This Project is structured in two parts:

  1. SQL Data Exploration
  2. Tableau Visualization

The primary objective of this project is to share a narrative about a devastating pandemic that claimed many lives. The purpose of this project is to visualize the data to enable all readers to comprehend and interpret the story. The story is based on data.

The dataset i'm using: Coronavirus (COVID-19) Deaths
The dataset version may change over time.

1. SQL Data Exploration

So the first thing we're going to do is explore the data with SQL. We are going to query the data because we will need it later for our tableau visualization.

Our Queries:

  • Global data
  • Total deaths per continent
  • Percent population infected per country
  • Percent population infected

2. Tableau Data Visualization

For our visualizations we use Tableau Public.
